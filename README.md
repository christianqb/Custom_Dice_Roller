# 🎲 Custom Dice Roller

Projeto desenvolvido com o objetivo de praticar **lógica de programação**, **estruturas de repetição** e **manipulação do DOM com JavaScript**.

A aplicação permite ao usuário definir quantos números deseja sortear e qual será o limite máximo desses números, simulando um sistema de sorteio personalizado.

---

## 🚀 Funcionalidades

* 🎯 Definir quantidade de números a serem sorteados
* 🔢 Definir limite máximo dos valores sorteados
* 🎲 Geração de números aleatórios
* 📦 Exibição dos resultados em formato de array
* ⚠️ Validação de entradas (evita valores inválidos)
* 🔄 Atualização dinâmica na tela

---
